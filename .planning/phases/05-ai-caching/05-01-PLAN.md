# Phase 5 Wave 1 — AI Caching Core Implementation

## Goal
Running `python scripts/run_pipeline.py` twice must produce zero OpenAI calls on the second run.
Every `Company` row must have a `description_hash` (64-char hex) after pipeline runs.

## Requirements
- CACHE-01: Pipeline skips company if `description_hash == compute_hash(description) AND industry IS NOT NULL`
- CACHE-02: `description_hash` is stored on each row after successful analysis
- CACHE-03: `scripts/migrate_add_hash.py` is idempotent — safe to run twice

## Tasks

### Task 1.1 — `scripts/migrate_add_hash.py` (new file)
```python
"""Idempotent migration: add description_hash column to existing company table."""
import sys
from pathlib import Path
sys.path.insert(0, str(Path(__file__).resolve().parent.parent))

from sqlalchemy import text
from sqlalchemy.exc import OperationalError
from app.database import engine

def migrate():
    with engine.connect() as conn:
        try:
            conn.execute(text("ALTER TABLE company ADD COLUMN description_hash TEXT"))
            conn.commit()
            print("Migration applied: description_hash column added.")
        except OperationalError:
            print("Migration already applied: description_hash column already exists.")

if __name__ == "__main__":
    migrate()
```

### Task 1.3 — `agent/analyzer.py` — add `compute_description_hash()` helper
Add at module level (no new imports — use `hashlib` from stdlib):
```python
import hashlib

def compute_description_hash(description: Optional[str]) -> Optional[str]:
    """SHA-256 hex digest of stripped description. Returns None for empty/None input."""
    if not description or not description.strip():
        return None
    return hashlib.sha256(description.strip().encode()).hexdigest()
```
No changes to `analyze_company()` — caching logic lives in the pipeline.

### Task 1.4 — `scripts/run_pipeline.py` — update skip logic and hash write
Replace the existing skip check:
```python
# OLD (industry only):
if company.industry is not None:
    skipped += 1
    continue
```
With two-condition check:
```python
# NEW (two-condition — CACHE-01):
computed_hash = compute_description_hash(company.description or "")
is_cache_hit = (
    computed_hash is not None
    and company.description_hash == computed_hash
    and company.industry is not None
)
if is_cache_hit:
    logger.info(f"[CACHE HIT] {company.company_name}")
    skipped += 1
    continue
```
After writing AI fields, also write the hash:
```python
company.description_hash = computed_hash
```
