**File structure — implement in this exact order:**

1. **Module docstring**: "YC JSON API scraper. Standalone — imports nothing from app/ or agent/."

2. **Imports** (stdlib only + requests):
   ```python
   import logging
   import sqlite3
   import time

   from typing import Optional

   ```

3. **Module-level constants**:
   ```python
   _DB_PATH = Path(__file__).resolve().parent.parent / "data" / "companies.db"
   _API_BASE = "https://api.ycombinator.com/v0.1/companies"
   MAX_PAGES = 2   # 2 pages × 25 companies = 50 records; sufficient for Phase 3 AI demo
   _DELAY_SECONDS = 0.5
   ```

4. **CREATE TABLE SQL constant** — must exactly mirror app/models.py column names and nullability:
   ```python
   _CREATE_TABLE_SQL = """
   CREATE TABLE IF NOT EXISTS company (
       id               INTEGER PRIMARY KEY AUTOINCREMENT,
       company_name     TEXT NOT NULL,
       description      TEXT NOT NULL,
       website          TEXT,
       industry         TEXT,
       business_model   TEXT,
       summary          TEXT,
       use_case         TEXT
   )
   """
   ```
   Cross-check every column against app/models.py before writing. The table name is `company` (SQLModel lowercases the class name). Do NOT add any column that isn't in models.py.

5. **`_ensure_table(conn: sqlite3.Connection) -> None`**:
   Execute `_CREATE_TABLE_SQL` and `conn.commit()`. Defensive idempotent call — Phase 1 creates this table via SQLModel, but the scraper is standalone and must not crash on a fresh DB.

6. **`_get_description(company: dict) -> str`**:
   ```python
   return (
       company.get("longDescription", "").strip()
       or company.get("oneLiner", "").strip()
       or f"{company['name']} (no description available)"
   )
   ```
   CRITICAL: `.strip()` before falsy check — whitespace-only strings ("  ") are truthy but semantically empty. Do NOT reference "shortDescription" — it does not exist in the YC API.

7. **`_upsert_company(conn, name: str, description: str, website: Optional[str]) -> None`**:
   ```python
   cursor = conn.execute("SELECT id FROM company WHERE company_name = ?", (name,))
   if cursor.fetchone():
       conn.execute(
           "UPDATE company SET description = ?, website = ? WHERE company_name = ?",
           (description, website, name),
       )
   else:
       conn.execute(
           "INSERT INTO company (company_name, description, website) VALUES (?, ?, ?)",
           (name, description, website),
       )
   conn.commit()
   ```
   NEVER use INSERT OR REPLACE — it deletes the old row and wipes AI fields (industry, business_model, etc.) populated in Phase 3.
   No UNIQUE constraint exists on company_name, so ON CONFLICT cannot be used.
   UPDATE touches ONLY description and website — never industry/business_model/summary/use_case.

8. **`fetch_companies(db_path: Optional[Path] = None) -> list[dict]`**:
   - Default: `db_path = _DB_PATH`
   - Call `db_path.parent.mkdir(parents=True, exist_ok=True)` before connect
   - Open `sqlite3.connect(str(db_path))` — wrap in try/finally to ensure `conn.close()`
   - Call `_ensure_table(conn)`
   - Create `requests.Session()` (enables TCP connection reuse)
   - Loop: `url: Optional[str] = _API_BASE`, `page_num = 0`
   - While `url and page_num < MAX_PAGES`:
     - `page_num += 1`
     - `resp = session.get(url, timeout=15); resp.raise_for_status()`
     - `data = resp.json()`
     - For each company in `data.get("companies", [])`:
       - Wrap in try/except Exception — log error and continue (one failure must NOT abort the batch)
       - `name = company["name"]`
       - `website = company.get("website") or None`  ← converts "" to None
       - `description = _get_description(company)`
       - `_upsert_company(conn, name, description, website)`
       - Append `{"name": name, "description": description, "website": website}` to stored list
     - `url = data.get("nextPage")`  ← None stops the loop
     - Insert `time.sleep(_DELAY_SECONDS)` ONLY if `url and page_num < MAX_PAGES` (not after last page)
   - Return stored list

9. **`main() -> None`**:
   ```python
   def main() -> None:
       stored = fetch_companies()
       logging.info("Done — %d companies upserted into %s", len(stored), _DB_PATH)
       print(f"Stored {len(stored)} companies into {_DB_PATH}")
   ```

10. **Entry point**:
    ```python
    if __name__ == "__main__":
        logging.basicConfig(level=logging.INFO, format="%(levelname)s %(message)s")
        main()
    ```
